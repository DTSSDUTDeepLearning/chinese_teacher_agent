# 古诗文默写出题 · 使用说明

## 一、功能简介

「古诗文默写出题」是为初中语文教师准备的自动化出题助手。你只需在对话框里用一句自然语言描述需求，系统就会自动从浙江省部编版初中语文教材的古诗文篇目中抽取句子，生成一份默写题，并附上答案与易错字解析，可直接复制到 Word / WPS 排版组卷。

**核心特点**：

- 对话式交互，零学习成本，像跟同事说话一样出题
- 题目原文来自本地知识库（非模型凭记忆背诵），确保原文准确
- 题目与答案分开输出，方便单独发放
- 易错字按需标注，有则标、无则不标，不制造噪音

---

## 二、适用场景

| 场景 | 说明 |
|------|------|
| 课堂练习 | 课前快速生成一份默写小测，课堂当场练习 |
| 课后作业 | 按本周所学年级学期出题，布置为家庭作业 |
| 期中/期末复习 | 指定整个年级或整个初中范围，综合复习出题 |
| 中考模拟训练 | 用"整个三年"范围出题，模拟中考默写题型 |

---

## 三、如何使用

系统主界面是一个对话窗口。在底部输入框里输入你的出题需求，按回车或点击发送按钮即可。系统会自动识别"出题"意图并路由到出题 Agent。

**触发关键词**：包含以下任意词语时，系统会走出题流程——

> 默写、出题、填空、理解性默写、上下句、古诗文、古诗、古文

如果不含这些词（比如只是闲聊），会走通用对话，不会出题。

---

## 四、可指定的出题参数

所有参数都通过自然语言描述，无需填写表单。未指定的参数会使用默认值。

### 1. 年级（篇目筛选维度）

| 可选值 | 示例说法 |
|--------|----------|
| 七年级 | "七年级"、"初一" |
| 八年级 | "八年级"、"初二" |
| 九年级 | "九年级"、"初三" |
| 全部 | "整个三年"、"全部"、"整个初中" |

### 2. 学期（篇目筛选维度）

| 可选值 | 示例说法 |
|--------|----------|
| 上册 | "上册"、"上学期" |
| 下册 | "下册"、"下学期" |
| 全部 | 不指定学期，或说"上下册都要" |

> **提示**：年级与学期是出题范围的唯一筛选维度。当前知识库覆盖七上至九上共 106 篇（七上 18 / 七下 17 / 八上 27 / 八下 21 / 九上 23）。

> **篇目范围说明**：出题篇目严格限定在上述教材知识库范围内，**不包含任何课外古诗词**，系统也不会提供"加入课外篇目"的选项。给定年级+学期后，系统自动在该范围内的全部篇目中抽取出题，教师无需也无法逐篇挑选。

### 3. 题型

| 题型 | 说明 |
|------|------|
| 混合（默认） | 一半上下句填空 + 一半理解性默写，打乱顺序输出 |

- **默认即为混合题型**，教师无需指定题型，系统也不会询问题型倾向。
- 如教师明确要求单一题型，可说"出理解性默写"或"出上下句填空"，系统按其指定出题；未明确指定时一律按混合题型出题。

### 4. 题目数量

- 默认 **6 道**（与中考题型一致）
- 可指定其他数量，如 "出 10 道"、"来 20 题"

---

## 五、使用示例

### 示例 1：指定年级学期 + 默认题型题量

```
帮我出6道八年级下学期的古诗文默写题
```

系统会生成八年级下册范围内的 6 道**混合题型**（3 道上下句填空 + 3 道理解性默写）。

### 示例 2：只说年级，系统会追问学期

```
出8道七年级的默写题
```

系统追问："请问您需要七年级上册、下册，还是都要？"

你补充："上册"，系统继续出题（混合题型）。

### 示例 3：整个初中范围（中考复习）

```
帮我出6道古诗文默写题，整个初中三年的范围
```

系统从七年级到九年级全部 106 篇中抽取出题（混合题型）。

### 示例 4：理解性默写（教师明确指定单一题型）

```
出6道九年级上册的理解性默写
```

系统为每道题生成一个语义描述题干（如"《岳阳楼记》中表达旷达胸襟的句子是______"），要求学生写出对应诗句。

### 示例 5：不指定任何条件

```
帮我出几道古诗文默写题
```

系统追问："请问您需要哪个年级、哪个学期的题目？（如：八年级下学期；或"整个三年"）"

## 六、输出说明

每份题目输出包含两部分：

### 第一部分：题目

```
1. ______，后天下之乐而乐。（《岳阳楼记》范仲淹）
2. 不以物喜，______。（《岳阳楼记》范仲淹）
...
```

- 数字编号，每题一行
- 上下句填空用 `______` 表示空缺处
- 理解性默写给出语义题干 + 空缺处
- 每题末尾标注出处（《篇名》作者）

### 第二部分：答案与解析

```
【答案与解析】
1. 先天下之忧而忧
   易错字："忧"不要写成"优"
2. 不以己悲
```

- 答案与题目编号一一对应
- **易错字按需标注**：只有当答案中存在易错字、易混淆字、生僻字时才标注；无易错字的题目不输出该行，避免噪音
- 易错字用引号标注，并给出辨析说明

### 复制使用

每条 AI 回复右下角有「复制」按钮，点击即可复制全部内容到剪贴板，再粘贴到 Word / WPS 中排版。

## 七、交互流程

```text
你输入需求
    │
    ▼
系统解析年级/学期/数量（题型默认混合，无需指定）
    │
    ├── 范围不完整 ──▶ 系统追问年级或学期 ──▶ 你补充 ──▶ 继续
    │
    ▼
从知识库检索对应篇目（全部篇目，无需逐篇挑选，不含课外）
    │
    ▼
按题型与数量抽取题目（默认混合，保证篇目多样性）
    │
    ▼
理解性默写题由 AI 生成语义题干
    │
    ▼
逐题匹配易错字（按需标注）
    │
    ▼
格式化输出：题目 + 【答案与解析】
```

## 八、使用小贴士

1. **范围说全更省事**：一次性说清"年级 + 学期"（如"八年级下学期"），可以避免系统追问，一次出题成功。
2. **"整个三年"是快捷说法**：中考复习时，直接说"整个三年"或"整个初中"即可覆盖全部篇目，无需逐个年级出题。
3. **题型默认混合**：系统默认生成混合卷（一半上下句填空 + 一半理解性默写），适合综合训练，无需指定。如需单一题型，明确说出即可。
4. **题量灵活**：默认 6 道与中考一致，但课堂小测可以少出几道（如"出 4 道"），综合练习可以多出（如"出 20 道"）。
5. **篇目多样性**：系统会尽量让每道题来自不同篇目；当范围内篇目较少时，才会出现同一篇目多题的情况。
6. **原文准确性**：题目原文来自本地知识库的标准文本，不是 AI 凭记忆背诵，可放心使用。如发现原文有误，欢迎反馈以便修正知识库。

## 九、常见问题

**Q：为什么系统有时候会反问我？**
A：当你说"出八年级的题"但没说学期时，系统无法确定是上册、下册还是都要，所以会追问。补一句"上册"或"下册"即可。

**Q：能按作者或朝代筛选吗？能只选部分篇目出题吗？能加课外古诗词吗？**
A：本期均不支持。当前只提供"年级 + 学期"两个篇目筛选维度，给定范围后系统自动在该范围全部篇目中抽取出题，教师无需逐篇挑选，也不能要求加入课外古诗词。这是与语文教师确认后的结果--教师实际出题时不会按体裁或重要性来筛篇目，也不希望被"选篇目"增加决策负担。按朝代/作者筛选已列入后续迭代。

**Q：题型是怎么确定的？需要我指定吗？**
A：不需要指定，系统默认生成混合题型（一半上下句填空 + 一半理解性默写）。如果你明确想要单一题型，说出"理解性默写"或"上下句填空"即可，否则一律按混合出题，系统不会追问。

**Q：理解性默写的题干是怎么生成的？**
A：系统先从知识库选出答案句，再由 AI 基于篇目全文生成语义描述题干，生成时会避免题干直接包含答案原句。

**Q：易错字数据从哪来？**
A：来自知识库中的预标注数据。目前并非每篇都有易错字标注，无标注的题目不会输出易错字行。后续会持续补充易错字数据。

**Q：题目内容会出错吗？**
A：原文来自本地知识库，准确性有保障。但理解性默写的题干由 AI 生成，建议教师核对后再使用。如发现任何错误，欢迎反馈。
