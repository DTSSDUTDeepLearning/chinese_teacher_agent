# 初中语文教师智能体 (Chinese Teacher Agent)

## 一、项目背景

本项目旨在为初中语文教师提供一个专注于日常教学场景的 AI 智能体助手。考虑到教育领域需求的多样性和渐进性，项目采用**"基于通用大模型 + 垂直领域封装"**的轻量级路线，优先快速产出核心教学辅助功能，并在实际使用中持续迭代完善。

## 二、项目定位

- **目标用户**：初中语文教师
- **核心思路**：不训练专有模型，而是基于通用大语言模型（LLM），通过提示工程（Prompt Engineering）、结构化输出和少量专业知识库，构建一个专注于初中语文教学场景的垂直领域智能体。
- **形态**：浏览器 Web 页面，采用对话式 UI，降低非技术用户的使用门槛。

## 三、核心功能规划

### 1. 古诗文默写出题（P0）

为初中语文教师自动生成古诗文默写填空题。教师通过对话式交互配置年级、学期、体裁、重要性等筛选条件，系统从对应范围中抽取句子，生成上下句填空题目。支持题目与答案分开输出，并可直接复制到 Word / WPS 排版组卷。

[详细需求文档](docs/requirements/feature-01-古诗文默写出题.md)

### 2. 课文背景与知识图谱（P1）

教师通过自然语言指令（如"查看《岳阳楼记》的作者生平"）触发，系统输出课文的作者生平、创作背景、文学流派等拓展信息，并以力导向图的形式展示"同作者"关联的知识网络。支持教师自定义关联关系（如"查看唐宋八大家"），动态生成新的知识图谱。适用于备课参考与课堂投屏展示。

[详细需求文档](docs/requirements/feature-02-课文背景与知识图谱.md)

### 3. 命题作文出题（P1）

教师与系统通过对话协商，共同"聊"出一份满意的作文题目。支持标准中考作文题（含材料、多话题、详细要求）与普通作文题两种形式，可控制文体、字数、难度、附加要求等维度。

[详细需求文档](docs/requirements/feature-03-命题作文出题.md)

## 四、建议技术路线

| 组件 | 建议方案 |
|------|---------|
| 大语言模型 | DeepSeek API（中文古诗文、作文任务表现优异，且成本可控） |
| 后端框架 | Python + `openai` 兼容 SDK 调用模型；可选轻量框架如 `pydantic-ai` |
| 交互界面 | 浏览器 Web 页面（对话式 UI，降低非技术用户使用门槛） |
| 输出格式 | Markdown / JSON，便于教师复制到 Word、PPT 等教学文档中 |

## 六、后续迭代方向

- 收集教师实际使用反馈，细化各功能的输入输出格式。
- 视情况引入本地知识库（如部编版初中语文课文清单、作者信息表），提升输出准确性。
- 探索多智能体协作架构，将备课、出题等任务拆分为更专业的子Agent。
