# 功能3：命题作文出题

## 一、功能概述

为初中语文教师提供命题作文的出题辅助服务，通过对话协商模式共同打磨出满意的作文题目。

## 二、作文题目形式

系统支持两种题目形式：

| 形式 | 说明 | 示例 |
|------|------|------|
| **标准中考作文题** | 包含材料、多个可选话题、详细写作要求 | 提供一段诗歌/名言材料，给出2-3个话题任选，要求题目自拟、不少于500字等 |
| **普通作文题** | 简洁明了，直接给出主题和要求 | "请以'文章与人品是否有相关性'为主题，写一篇不少于500字的作文" |

## 三、触发方式：对话式出题

- **非一次性生成**，而是采用**对话协商**模式。
- **流程**：
  1. 教师输入主题，并声明题目类型（中考题 / 普通题）。
  2. 系统根据主题生成初稿题目。
  3. 教师可提出修改意见（如"换个材料""增加一个话题选项""字数要求改为600字"等）。
  4. 系统根据反馈迭代优化，直至教师满意，确认最终版本。

## 四、出题维度

教师在对话过程中可控制的变量：

| 维度 | 说明 |
|------|------|
| 文体 | 记叙文 / 议论文 / 说明文 / 不限（初中以记叙文为主） |
| 字数要求 | 如"不少于500字""600字左右""800字以上"等 |
| 难度 | 七年级水平 / 八年级水平 / 九年级水平 / 中考模拟难度 |
| 附加要求 | 如"要求运用景物描写""要求有细节描写""要求写出真情实感"等 |

## 五、输出内容

系统最终输出**作文题目本身**，包含：

- **材料**（中考题形式）：引用的诗歌、名言、故事等材料。
- **写作任务/话题**：可供选择的标题或话题方向。
- **基本要求**：字数、文体、不得套作抄袭等常规要求。

> **不输出**：写作提示/导语、评分标准、范文、写作提纲等。

## 六、最小可行版本（MVP）

第一轮实现以下功能：

1. 教师输入主题和题目类型，系统生成初稿题目。
2. 支持多轮对话修改，直至确认最终版本。
3. 支持控制文体、字数、难度、附加要求等维度。

## 七、后续迭代方向
