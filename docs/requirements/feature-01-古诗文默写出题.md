# 功能1：古诗文默写出题

## 一、功能概述

为初中语文教师自动生成古诗文默写填空题，用于课堂练习、课后作业或考试组卷。

## 二、出题范围

- **核心范围**：浙江省部编版初中语文教材中的古诗文篇目。
- **扩展范围**：支持教师手动输入诗文标题、全文或常考名句作为补充来源。

## 三、筛选条件

教师在出题前需配置以下筛选维度：

| 筛选维度 | 选项 | 说明 |
|----------|------|------|
| 年级 | 七年级 / 八年级 / 九年级 / 全年级 | 全年级模式适用于中考综合复习 |
| 学期 | 上册 / 下册 / 全部 | 与年级组合筛选 |
| 体裁 | 诗词 / 文章 / 全部 | 诗词通常考频更高，文章考频较低 |
| 重要性 | 全部 / 仅重点篇目 | 重点篇目指具有重大历史意义或文学地位的名篇（如《岳阳楼记》《出师表》） |

> **暂不支持（第二轮迭代）**：按朝代/作者筛选、按历史考卷出题频率筛选。

## 四、题型设计

**唯一题型：上下句填空**

- 给出上句，要求填写下句。
- 或给出下句，要求填写上句。

**示例**：
- `"_______"，不尽长江滚滚来`（答案：`无边落木萧萧下`，出处：《登高》）
- `飘飘何所似，"_____"`（答案：`天地一沙鸥`，出处：《旅夜书怀》）

> **暂不支持**：理解性默写、错别字辨析、选择题等其他题型。

## 五、输出格式

系统输出包含以下内容：

1. **题目列表**：仅含填空题，标注课文名。
2. **答案列表**：独立输出，与题目一一对应，供教师参考或单独发放。
3. **复制友好**：支持一键复制到 Word / WPS 进行排版组卷。
4. **在线展示**：支持在 Web 页面直接展示，可设置学生作答模式（隐藏答案）。

## 六、重要性分级标准（初版）

系统按以下粗略等级对课文进行重要性划分：

| 等级 | 描述 | 示例 |
|------|------|------|
| S级 | 千古名篇，中考高频，必背 | 《岳阳楼记》《出师表》《陋室铭》 |
| A级 | 经典诗词，常考 | 李白、杜甫、苏轼等名家代表作 |
| B级 | 课内要求背诵但考频较低 | 部分游记、小品文、骈文 |

> **注意**：第一轮由模型按常见教学共识自动分级，教师可在使用过程中手动调整。

## 七、准确性策略

- **第一轮采用方案 A（纯模型生成）**：依赖 DeepSeek 的背诵能力，不引入本地知识库校验。
- **风险应对**：
  - 在输出页面附加免责声明："题目由 AI 自动生成，请教师核对后使用。"
  - 鼓励教师和学生发现错误后及时反馈，用于后续迭代优化。

## 八、最小可行版本（MVP）

第一轮实现以下功能：

1. 教师选择年级、学期、体裁、重要性等级。
2. 系统根据条件从对应范围中随机抽取句子。
3. 生成上下句填空题目，标注课文名。
4. 分开输出题目和答案。
5. 支持复制到 Word / 网页展示。
