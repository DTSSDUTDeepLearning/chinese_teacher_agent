# 部署环境安装指南

本指南说明在目标运行环境（Windows 操作系统）中，部署和运行本项目所需的全部软件依赖及配置要求。

**核心原则**：最小化目标环境的软件安装负担，降低非技术用户的使用门槛。

---

## 一、目标运行环境

### 1.1 环境假设

- 操作系统：Windows 10 / Windows 11
- 目标用户：非技术开发人员，无 Python、Node.js 等开发环境基础
- 网络环境：可访问互联网（用于下载安装包及调用 DeepSeek API）

### 1.2 软件安装清单

目标环境仅需安装以下两款软件：

| 软件名称 | 推荐版本 | 用途 | 官方下载地址 |
|----------|----------|------|-------------|
| Python | 3.11.9 | 运行后端 FastAPI 服务 | https://www.python.org/downloads/release/python-3119/ |
| MySQL Community Server | 8.0.37 | 数据持久化存储 | https://dev.mysql.com/downloads/installer/ |

> **安装提示**：
> - Python 安装时务必勾选 "Add Python to PATH" 选项。
> - MySQL 安装时选择 "Server only" 模式即可，无需安装 MySQL Workbench 等图形化管理工具。
>
> 详细安装步骤请参阅：
> - [Python 安装指南](python-installation-guide.md)
> - [MySQL 安装指南](mysql-installation-guide.md)

---

## 二、启动流程

目标环境完成上述两款软件安装后，按以下步骤启动应用：

1. 获取项目分发包（压缩文件或离线拷贝）。
2. 解压至本地目录。
3. 运行项目根目录下的启动脚本（`start.bat`）。
4. 脚本将自动完成以下操作：
   - 创建 Python 虚拟环境
   - 安装项目依赖（`requirements.txt`）
   - 初始化数据库表结构
   - 启动后端服务
5. 打开浏览器，访问 `http://localhost:8000` 进入应用界面。

---

## 三、开发环境要求

开发环境需在目标运行环境的基础上，额外安装以下软件：

| 软件名称 | 推荐版本 | 用途 |
|----------|----------|------|
| Node.js | 20.14.0 LTS | 前端 Vue 3 项目的开发、构建与打包 |
| Git | 最新稳定版 | 源代码版本管理 |
| VS Code | 最新稳定版 | 代码编辑器（推荐） |

---

## 四、云服务器部署（后续可选）

若后期采用云服务器部署（如阿里云 ECS、腾讯云 CVM），则：

- **目标用户端**：无需安装任何软件，通过浏览器直接访问部署地址即可使用。
- **服务器端**：安装 Python 3.11 + MySQL 8.0，配置与本地环境一致。
- **部署方式**：在开发环境中完成前端构建，将构建产物与后端代码一并部署至云服务器。

## 五、版本兼容性说明

| 组件 | 最低版本 | 推荐版本 | 备注 |
