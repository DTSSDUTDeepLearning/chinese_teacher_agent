## 核心决策
- **项目定位**：面向全体初中语文教师的垂直领域教学智能体。
- **技术路线**：
  - 大模型：DeepSeek API
  - 后端：Python + FastAPI + `openai` SDK
  - 前端：Vue 3 + Ant Design Vue
  - 数据库：MySQL
  - 输出格式：Markdown / JSON
- **功能优先级**：
  1. 古诗文默写出题 (P0)
  2. 课文背景与知识图谱 (P1)
  3. 命题作文出题 (P1)
